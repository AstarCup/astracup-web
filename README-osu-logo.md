# osu! Logo Setup

For the authentication system to work properly, you need to add the osu! logo:

1. Download the official osu! logo from: https://osu.ppy.sh/images/logo-osu.png
2. Save it as `public/osu-logo.png`
3. Alternatively, you can use a CDN URL in the sign-in page

The sign-in page expects the logo to be available at `/osu-logo.png`.
