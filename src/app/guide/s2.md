# AstaraCup 星域杯 S2 比赛手册

## 赛事介绍

AstaraCup 是面向中高分段 osu!std 玩家的娱乐性比赛。本赛季（S2）采用全新的**多人场次轮次淘汰赛**(吃鸡)，选手通过提名谱面与统一的 Stand / Random 图池，在多轮比赛中争夺最终冠军。比赛使用 **lazer** 客户端。

## 时间安排

| 阶段               | 日期             | 说明                         |
| ------------------ | ---------------- | ---------------------------- |
| 报名               | x月x 日 – x月 x日(7day) | 选手报名，选择可参加场次     |
| 公布场次 & 提名谱面 | x 月 1 日 – x月x 日(7day) | 分配场次，选手提交提名谱面   |
| 比赛               | x月x日 – x 月 x日(30day) | 各场次比赛，直至决出冠军     |

## 报名规则

1. **PP 要求**：报名时 **8500 ≥ pp ≥ 5500**，以官方显示的当前 pp 为准（若曾因违规被清除 pp，则按历史最高 pp 计算）。
2. **参赛资格**：必须已在比赛 QQ 群内；仅限 **中国大陆玩家**（港澳台地区获奖需自付奖品邮费）；账号状态正常，无长期离线、近期不良游玩记录。
3. **时间选择**：报名时需勾选自己能参加的**场次段**。
4. **账号审查**：工作人员将对报名账号进行合理性、游玩记录、有无代打/小号等检查，不合格者取消资格。
5. **退赛**：报名截止前可自由退赛；进入赛群后又退出，视为弃赛。

## 场次分配

- 报名结束后，主办将根据报名时提交的**pp** 进行分场。同一场次内选手的 pp 差距将尽量控制，保证竞技平衡。
- 若某场次人数不足，可能会安排 pp 较低但时间可匹配的选手加入。
- 每场次人数为 **2~16 人**。具体淘汰人数、图池星数范围等细节可能根据场次规模微调，将在场次名单公布时一并说明。

## 选手提名谱面

1. 场次名单公布后，每位选手需在 **x月 x 日前提名一张谱面**，作为该场次的**选手谱面图池**（不对外公开）。
2. 提名谱面必须满足 [谱面要求](#谱面要求) 中的所有条件。审核未通过时，Staff 会私聊通知修改；逾期未改则由主办代为选择（不保证完全符合所有要求）。
3. 所有到场选手的提名谱面共同组成该场次的私有图池，供 Random 环节抽取。

## 谱面要求

为保证公平与可玩性，所有比赛用图（包括提名谱面与图池 Stand 谱面）必须满足：

- **谱面状态**：官网可搜索并下载的 osu!std Ranked 或 Qualified 谱面。
- **星数**：星级在 **0.1★ – 7.5★** 之间（不同场次具体范围可能更严格，以场次公告为准）。
- **长度**：总时长 ≤ **5:00**。
- **参数限制**：若星数 > 3.0★，则 **AR ≥ 8** 且 **CS ≤ 6**。
- **Mod 规则**：所有图默认视为 **Freemod**，不可强制指定特定 Mod（如 DT、HT 等）；选手可自行开启合法 Mod 组合(包括Lazer fun mod: **TC** )，但 **禁止使用 DT 、 HT** 、**EZ** 。
- 比赛全程强制开启 **No Fail (NF)**。

## 赛制详解

### 基本流程

- 比赛开始前 **15 分钟**，裁判创建多人房间；**10 分钟**前向每位选手发送邀请。
- 若比赛规定时间 **10 分钟后**仍未进入房间，该选手视为弃权。
- 到场选手到齐或比赛规定时间 **15 分钟后**，比赛正式开始（取较早者）。

### 轮次与淘汰

- 根据场次当前剩余人数，比赛可能分为 **若干轮次**。
- 每一轮中，选手按 **图池顺序从上到下** 依次游玩所有谱面。
- 图池分为两种类型：
  - **Stand**：主办预设的固定谱面，所有选手提前可知。
  - **Random**：从选手谱面图池中 **随机抽取一张**，抽中时公布选图者。已抽过的谱面不会在本轮再次被抽。
- 每张图结束时，立刻根据排名计算该图得分（不淘汰）。
- **本轮所有图打完后**，结算总得分，排名**最末的 X 位选手**被淘汰。剩余选手进入下一轮，**积分重置**。
- 重复进行轮次，直至最后仅剩 **1 名**选手，即为本场次冠军。

### 得分与排名

- 单图得分 = **X - Y + 1**  
  *X = 本轮次开始时的参赛总人数，Y = 选手在该图的排名（如第 1 名 Y=1）。*
- 失利（Fail）则该图得 **0 分**。
- 同分规则：
  1. 比较本轮所有图的 **得分总和**，高者列前；
  2. 若仍相同，则由裁判根据选手在本轮的整体表现给出 **印象评分**，决定先后。

### 重赛与掉线

- 开赛后 **30 秒内**，因设备/网络原因可申请重赛，由裁判酌情裁定。超过 30 秒后不可重赛。
- 每轮比赛每位选手最多可允许 **2 次** 掉线。第 3 次起，该图成绩记为 0 分。
- 成绩上传失败时，可提供 **未经篡改的录屏/截图/回放** 作为有效成绩凭证。

### 其他通用规则

- 比赛中出现争议请第一时间联系 Staff，禁止与选手发生争执。
- 恶意拖延时间、不服从裁判指挥者，裁判有权取消其当场成绩或参赛资格。
- Staff 若符合参赛条件，可报名参加**不同的场次**。
- 不同场次的具体细则（如图池具体内容、淘汰人数 X 等）将在各自场次公告中提前公布，请以当场规则为准。
