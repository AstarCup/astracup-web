# AstraCup 星域杯 S1 比赛手册

# 赛事介绍

AstarCup是面向中高分段的一个娱乐性比赛，其目的是主办起星域了。比赛会办成系列赛，预计一年一届。本系列赛事模式为osu!std。使用**lazer**端进行比赛。

本赛事中，为个人淘汰赛赛制，**不设置FM图池，改为lazer mod图池**。

比赛途中出现问题请找staff ，不要与玩家发生争执。

# 比赛常规介绍

- 比赛中默认使用计分方式为 **lazer score** ，默认**不**开启No Fail (NF) ，场上选手**会**因为失败而结束比赛。
- 每场比赛开始前**5**分钟，该局裁判将邀请双方进入多人游戏房间。进入mp房间进行热手后开始比赛并进行bp环节。
- 比赛流程大致为：**进场〉热手〉双方bp〉打图〉另一方pick图〉打图...** 直到一方达到胜利要求，本场比赛才会宣布结束。

# 报名

- 本次报名要求pp段的区间为**2000pp - 8100pp** 。在这区间的玩家均可以报名本赛事。
- 本次赛事pp限制以报名时的pp作为基准。
- osu账号未处于长期离线状态的，近期无不良游玩记录的方可报名。
- 如果开赛后在赛中被查到作弊行为的，将会取消本届的参赛资格，并对该账号进行举报，并禁止该账号参与本系列赛事。
- 请确保有充足时间参与，如果没有可以在截止报名前允许退赛。
- 已报名但仍未进入赛群的玩家将视为弃赛，报名后参与且退出赛群的玩家将视为弃赛。
- 报名完毕后工作人员将会进行账号检查，检查不合格将被限制参赛。

## 检查内容包括但不限于：
- 账号合理性
- bp[最好成绩]
- etx的LR（League Rating）
- 游玩记录和时长
- 有无代打/小号/多账号的违规行为

# 赛制和对阵

## 比赛赛制

- 若达成8人，则进入单败淘汰赛。图池为2轮。
- 若达成16人，则采用 资格赛+双败 赛制。图池为 4/5/6轮[包括资格赛图池]。

## 赛前对阵时间

- 推荐时间段为工作日17:00 ~ 21:00，节假日/周末10:00 ~ 22:00。
- 若双方无法协商出合适的比赛时间段，可以推迟比赛时间，但不得超过下一周比赛的日程安排。
- 比赛开始前**3**小时不可以更改时间。

## 热手

- 比赛前双方直接选图你需要的热手图。
- 热手阶段默认先游玩红方的热手图。
- 使用的热手谱面时长应小于或等于300s(5分钟)，星级不得超过**8**星。
- 热手结束后，双方需要在游戏内进行roll点以决定bp顺序。
- 如果对手迟到的情况下，双方无热手直接开始比赛。
- Roll点需要**群机器人进行roll点**。（没banchobot是这样的）

## 获胜条件和赛制

其中一方达到胜局数时取得该场比赛胜利。当双方对决至当前回合数上限时，将会进入决胜局（TieBreaker），且强制使用图池内的 TB 作为决胜图进行对局。

## 资格赛阶段

当报名人数大于16人时，将会通过一轮图池决出总平均排名更高的16位选手作为双败淘汰赛的参赛选手，资格赛低于16名的玩家将会在资格赛中淘汰。

### 平均排名规则

- 总平均排名决定顺位进入双败淘汰赛。
- 排名根据玩家总平均分数进行排名。
- 如果出现平均分一样的情况下，进行群roll点后排序。

## 双败阶段

- 一 轮图池为 BO9[抢5]。
- 二 三轮图池为 BO11[抢6]。
- 四 五轮为 BO13[抢7]。
- 本届比赛**没有**总决赛的复活甲 (bracket reset)。[这里指败者组不需要通过连续赢下两场比赛才能获得胜利的情况]。

# 比赛规则

- 在确定好的比赛开始前**5**分钟，裁判会创建对应的多人游戏房间。选手需要在这个时间段内进入房间准备比赛。
- 如果比赛开始时，对手还没到场，比赛可以推迟10分钟。如果推迟后还不在线，则判对方默认获胜 (Win by default)。
- 每方将有120s的时间选择谱面，以及 120s的准备时间。
- 准备时间耗尽后裁判可以选择适当加长时长。
- 可以申请延时 [一次延时3min ] 双方各2次机会（如果一次申请2个延时，则第2次只能+1min）。
- 双方准备完毕时 10s后开启比赛。
- 如果某个对局出现了平局 (两人分差为0)，则该局成绩作废，重赛一局作为该图结果。
- 一场比赛中多次无理由超时视为蓄意拖延比赛时间，警告后如果还是一样将判负处理。
- 比赛开始后 30 秒以内，可以请求重赛。裁判可以根据情况允许或者拒绝这次重赛请求。**超过 30 秒后不可以申请重赛**。
- **每场比赛**掉线只有2次机会，第3次及以上掉线视为本次成绩无效[0分], 判对方1分。

## BP环节[Ban/Pick]

- Pick - 选择此赛图，Ban - 禁用此赛图。
- 本届赛事能在图池中挑选并禁用一张图 (1 Ban)。被禁用的图在该场比赛中都不能使用。
- 在比赛开始后使用**群roll**命令来投点数。点数大者可选择先ban后pick 或 后ban先pick。
- 每次bp时间限制为120秒，如果ban超时则空ban，如果pick超时则裁判代pick。
 
## 图池与奖励

### 图池组合 

在本次赛事图池结构中的图池组合分为**NM HD HR DT LZ TB**
各部分图池含义如下：
NM [No Mod] : 使用 NF[NoFail] mod.
DT [DoubleTime] : 使用DT和NF mod.
HD [Hidden] : 使用HD和NF mod.
HR [Hard Rock] : 使用HR和NF mod.
TB [Tiebreaker] : 进入双方赛点时才会被强制使用，必须开启NF. TB的mod不受限制。
LZ [Lazer]: lazer图，使用lazer fun mod内精心挑选的图，不可ban。
